Download Path

Path has gone open source in two releases and builds are distributed through the Unity asset store.

So you have a couple of options for getting a copy of Path:

  • Path 2 build (AngryAnt) on Unity Asset Store.
  • Path 2 source (MIT) on github.
  • Path 1 source (GPL) on github.
